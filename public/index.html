Микросервис Api

Примечание: Заголовок HTTP-запроса - это информация в виде текстовой записи, которую браузер пользователя отправляет на веб-сервер содержащий подробную информацию о том, что браузер хочет и будет принимать обратно с сервера. Заголовок запроса также содержит тип, версию и возможности браузера, который выполняет запрос, чтобы сервер возвращал совместимые данные. После получения заголовка запроса сервер вернет подключенному клиенту заголовок HTTP-ответа к отправляемым файлам.

Пример Использования:

[base url]/api/KTO?

Пример Вывода:

{"ipaddress":"::3000:127.0.0.1","language":"ru-RU,ru;q=0.5",
"software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Firefox/50.0"}
